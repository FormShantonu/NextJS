### Difference between Reat and Next js

* React : 
    - Routing needs to external lib(React router dom)
    - Rendering CSR(Client Side Rendering)
    - Api handeling needs separate backend setup
    - Performance browsers does all the work
    - Production ready need extra setup

* Next js:
    - Build in file baseed routing
    - Rendering CSR + SSR
    - Api handeling build in api router
    - Performance optimized by default
    - Production ready out of box
